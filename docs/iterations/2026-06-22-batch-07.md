# 第 7 批：个人中心、后台看板与交付加固

## 实施内容

- 个人中心聚合救助、领养、志愿者、认捐、活动和通知记录。
- 后台首页增加动物、待审救助、有效领养、志愿者、捐赠和活动数据。
- FAQ 通过页面布局统一展示，业务页面只需提供当前模块数据。
- 动物、救助、海报和社区图片统一限制 5MB，志愿者材料限制 10MB。
- 补充个人数据隔离、后台业务统计和文件限制测试，更新部署媒体卷说明。
