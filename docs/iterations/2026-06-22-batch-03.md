# 第 3 批：领养中心闭环

## 实施内容

- 新增领养申请和领养关系，约束重复待审申请与单动物有效关系。
- 新增事务化提交、审批和关系终止服务，审批时重新校验动物与用户名额。
- 用户当前有效领养最多 3 只；通过一份申请后自动驳回竞争申请。
- 新增领养列表、详情、申请表单和个人领养记录。
- 新增 SimpleUI 领养菜单，申请与关系数据不允许后台任意新增或删除。
- 同步 Django migration、MySQL 8 参考 SQL 和行为测试。
