# 第十五批：后台管理优化

## 变更

- 使用 SimpleUI 自定义菜单完善用户列表、新增用户和批量新增入口。
- 新增 ECharts 用户数据看板，展示用户总量、状态、后台人员和六个月增长趋势。
- 完善用户列表筛选、搜索、头像、批量启停及新增/编辑表单视觉。
- 新增事务化 CSV 批量用户创建，限制 1MB、500 行并提供逐行校验反馈。
- 新增 `agent-docs/admin.md`，约束后台配置、权限、模板、批量操作和验证方式。
- 按 Product Design 选定的“全宽分区”方案重构用户新增和编辑表单：取消错位卡片，统一双栏字段网格、帮助文本、文件上传、权限选择器与底部操作栏。
