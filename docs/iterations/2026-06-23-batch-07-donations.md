# 第 7 批（捐赠模块）：暖橙主题重构

## 实施内容

### CSS 重构（`static/donations/css/donations.css`）
新建暖橙主题捐赠样式：

1. **页面头部** — `section-kicker` + 大标题 + 副标题 + 行动按钮
2. **捐赠项目卡片** — 3 列网格（平板 2 列，手机 1 列），白色卡片 + 阴影 + 悬停上浮；状态标签使用浅橙背景；描述截断 3 行
3. **捐赠详情页** — 项目描述 + 需求物资列表；每物资双栏布局（左信息 + 右进度/表单）
4. **进度条** — 暖橙渐变填充，轨道为浅灰；JS 动态计算已完成百分比（`data-remaining` / `data-required`）；标签显示剩余量/总量
5. **认捐表单** — 数量输入框（窄）+ 备注输入框（弹性）+ 认捐按钮，内联布局；移动端堆叠
6. **我的认捐** — 行列表（左项目信息 + 右状态标签），悬停左移反馈
7. **分页器** — 药丸按钮 + 页码显示
